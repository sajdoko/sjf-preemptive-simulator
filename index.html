SJF Preemptive Scheduler

Process List

    Result:

      SJF Preemptive Algorithm

      1. Initialize an empty list of processes (queue)
      2. While there are processes in the queue:
      3. Sort the queue by burst time
      4. Select the process with the shortest burst time from the front of the queue
      5. Assign the CPU to the selected process
      6. Decrement the burst time of the selected process
      7. If the burst time of the selected process is 0, remove it from the queue
      8. If a new process arrives during the execution of the selected process:
      9. If the burst time of the new process is shorter than the remaining burst time of the current process, interrupt the current process and execute the new process
      10. If the CPU is idle, go to step 2
